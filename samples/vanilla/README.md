# WOFF Starter
Sample code of WOFF Starter app made with no frameworks (vanilla).

## Usage
### Workspace with Docker
```sh
docker compose up -d
docker compose exec workspace bash
```

### Prepare

```sh
export WOFF_ID=WOFF_ID
npm install
```

### Run local server

```sh
npm run serve
```

### Build

```sh
npm run build
```
